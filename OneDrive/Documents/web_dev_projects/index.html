Introduction to Web Development

Web development is the process of creating websites and web applications. It involves a combination of programming, design, and content creation to build functional and visually appealing websites.

Key Terminologies of Web Development

Terminologies Descriptions
1. World Wide Web World Wide Web: An information system where documents and other web resources are identified by URLs, interlinked by hypertext, and accessible via the internet.
2. Transmission Control Protocol/Internet Protocol Transmission Control Protocol/Internet Protocol (TCP/IP): The foundational suite of communication protocols used to interconnect network devices on the internet, ensuring data is delivered reliably and to the correct destination.
3. Web Browsers Web Browsers: Software applications, such as Chrome or Firefox, that retrieve and render content from the World Wide Web for users to view and interact with.
4. Web Servers Web Servers: Computers or software that store web files and deliver them to users' browsers via the internet when a request is made.
5. Uniform Resource Locators Uniform Resource Locators(URL): The specific web address used to locate a unique resource, such as a webpage or image, on the internet.
6. Domain Name Server Domain Name Server(DNS): The internet's "phonebook" that translates human-readable domain names like google.com into the numerical IP addresses computers use to identify each other.
7. Hypertext Transfer Protocol Hypertext Transfer Protocol(HTTP): The standardized application-layer protocol used for transmitting hypermedia documents, serving as the basis for data communication on the World Wide Web.
8. Intranet Intranet: A private network accessible only to an organization's staff, used for secure internal communication and collaboration.
9. Extranet Extranet: A controlled private network that allows external partners, vendors, or authorized customers to access specific parts of an organization's intranet.
10. Multitier Architecture Multitier Architecture: A client-server architecture in which presentation, application processing, and data management functions are physically separated into distinct layers or tiers.
11. File Transfer Protocol File Transfer Protocol(FTP): A standard network protocol used for the direct transfer of computer files between a client and a server on a computer network.
12. Hypertext Mark-up Language Hypertext Mark-up Language(HTML): The standard markup language used to create and structure the content and layout of pages displayed on the World Wide Web.
13. Web Versions
  • Web 1.0: Static webpages (Read-only).
  • Web 2.0: Social/Participatory web (Read-Write).
  • Web 3.0: Semantic/Decentralized web.
  • Web 4.0: Symbiotic/AI-driven web.
